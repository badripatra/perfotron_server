Installation

Pre-requisites: git , python2.7

Suppoted Env: Unix (4 Core , 8 GB RAM)

Steps :

  1. Login as sudo user to your unix/linux box where you would like to install PerfoTron
  2. Update your system with "sudo apt-get update" or "sudo yum update" based on unix / linux flavor
  3. Clone the Repo - "git clone https://badri_patra@bitbucket.org/badri_patra/perfotron.git"
  4. Start the setup - "python perfotron"
  5. Select 'Yes' or 'No' to proceed after reading Tool Usage and Instllation details
  6. Access the Grafana URL printed on console for Live Monitoring of Load Test
