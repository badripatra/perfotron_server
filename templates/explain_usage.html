Monitor Load Test run from PerfTron Dashboard :

The dashboard has 5 sections :

  • Header
  • Summary
  • Errors
  • Individual Transaction Details
  • PerfoTron Server Stats

Details on Header section:

  • application - Enables users to select AUT (Application Under Test) to see related graphs
  • Transaction - Enables users to select a specific request type. This populates data in 'Individual Transaction Details' section
  • server - Displays Perfotron server name
  • Clock - India Time/ PST Time
  • Received / Sent Bytes - Total bytes sent / received for the Load Test
  • Over All Error Count - Total Errors (HTTP + Assertion Errors)
  • Error Indicator - Indicates  Errors are in  Acceptable / Non Acceptable range based on threshold set. (Acceptable <= 7 % , Not Acceptable > 7%) 
  • Overall Response Time - Average response time of all transaction in milliseconds
  • Response Time Indicator - Indicates  average response time of all transactions are in  Acceptable / Non Acceptable range based on threshold set (With in SLA  = 0 - 3000 ms, Breaching SLA = 3000 - 5000 ms, High Response > 5000 ms )

 

Details on Summary section(Graphical):

  • Success Rate %- Success Percentage of requests
  • Error Rate % - Failure Percentage of requests
  • Total Requests - Total requests
  • Failed Requests - Failed requests
  • Total Throughput - Number of requests per second
  • Total Errors - Number of Errors occurred (HTTP and Assertion Points)
  • Active Users - Number of Threads
  • Transaction Response Time Average - Average response of each type fo request in milli seconds
  • Stats - Response Time(Min/Max/Average/90th Percentile) in secs and Received Bytes
  • Network Traffic Sent Bytes - Bytes sent during load test
  • Network Traffic Received Bytes - Bytes received during load test

Details on Errors section(Graphical):

  • Errors per Transaction - Count of error for each type of request
  • Error Info - Error details like response code, response message,  request type, total count

Details on Individual Transaction section(Graphical):

  • Success Rate %- Success Percentage of request type selected
  • Error Rate % - Failure Percentage of request type selected
  • Total Requests - Total requests made of request type selected
  • Failed Requests - Failed requests made of request type selected
  • Total Throughput - Number of requests per second of request type selected
  • Total Errors - Number of Errors occurred (HTTP and Assertion Points) request type selected
  • Active Users - Number of Threads of request type selected
  • Transaction Response Time Average - Average response in milli seconds of request type selected
  • Network Traffic Sent Bytes - Bytes sent during load test of request type selected
  • Network Traffic Received Bytes - Bytes received during load test of request type selected

Details on PerfoTron Server stats section(Graphical):

  • Disk Usage % - Percentage of Disk space available (Identified using df command)
  • CPU Used% - Percentage of CPU used (Identified using top command)
  • Available Memory - Memory(RAM) left in MB (Identified using free command)
  • USED Memory - Memory(RAM) used in MB (Identified using free command)
