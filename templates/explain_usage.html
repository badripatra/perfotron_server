PerfoTron(Load & Performance Testing Platform)

  1. Convert and run your existing jmeter script in PerfoTron server by using --jmx option - "python perfotron --jmx <Jmeter_TestPlan_Complete Path>"
  2. The converted jmx & test results are is stored in '~/user_jmx_run_results/<Current_Date_Time_Stamp_Folder>
  3. You can do live monitoring @  "http://[perfoTron_server_ip]:3000/d/sample_dashboard/performance-dashboard?orgId=1"

 

                                                  OR

  1. Convert your jmx from your desktop / laptop using convert_jmx link
  2. Trigger the test from your Local System Jmeter
  3. You can do live monitoring @  "http://[perfoTron_server_ip]:3000/d/sample_dashboard/performance-dashboard?orgId=1"

 

How to interpreat the Perftron Dashboard :

The dashboard has 5 sections :

  • Header
  • Summary
  • Errors
  • Individual Transaction Details
  • PerfoTron Server Stats

Details on Header section:

  • Clock - India Time/ PST Time
  • Received / Sent Bytes - Total bytes sent / receveid for the Load Test
  • Over All Error Count - Total Errors (HTTP + Asserstion Errors)
  • Error Indicator - Indicates  Errors are in  Acceptable / Non Acceptable range based on threhold set. (Acceptable <= 7 % , Not Acceptable > 7%) 
  • Overall Response Time - Average response time of all transaction in milliseconds
  • Response Time Indicator - Indicates  average response time of all transactions are in  Acceptable / Non Acceptable range based on threhold set (With in SLA  = 0 - 3000 ms, Breaching SLA = 3000 - 5000 ms, High Response > 5000 ms )
