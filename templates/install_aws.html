PerfoTron(Load & Performance Testing Platform)

Pre-requisites:

  • git
  • python 2.7
  • Knowledge on 'Creating EC2 Instances in AWS'
  • ec2 instance(t2.large)
  • aws cli installed and configured Reference Link

Installation Steps :

  1. Login to your cloud instance where you would like to install PerfoTron
  2. Switch to sudo user
  3. Update your system with "sudo apt-get update" or "sudo yum update" based on unix / linux flavor
  4. Clone the Repo - "git clone https://badri_patra@bitbucket.org/badri_patra/perfotron.git"
  5. Start the setup - "python perfotron"
  6. Select 'Yes' or 'No' to proceed after reading Tool Usage and Instllation details
  7. Access the Grafana URL printed on console for Live Monitoring of Load Test

Note : PerfoTron will open 3000, 8086, 9003 ports during installation. You can open this ports manually incase, you wish to before installtion takes place and tag it your instance.
