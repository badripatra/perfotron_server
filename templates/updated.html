Overview

The Tool offers E2E Platform for Performance Testing by integrating Jmeter with Influx DB and Grafana.

  • Target Audience       : Individual / group using Jmeter for Performance Testing
  • How it helps          : Live Monitoring of Performance Tests triggered by Jmeter
    • Grafana Version       : 6.5.2
  • Influx Version        : 1.7.4
  • Jmeter Version        : 5.1.1
  • Programming Language  : Python 2.7, Shell Scripting
  • Supported Envs        : Unix (Ubuntu,CentOS, RHEL) for Hosting
