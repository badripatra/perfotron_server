  PerfoTron provides a high class open source Platform for monitoring Load and Performance Tests for APIs, Web based Applications.

  Built with a objective to introduce performance testing at every stage of application development.

  E2E Platform is developed using Jmeter,Influx DB, Grafana, Python, Shell Scriting.
