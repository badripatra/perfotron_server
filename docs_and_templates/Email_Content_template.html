Details of Report :

consolidated teststats csv - 

  • avg-rt - average response time in seconds
  • avg-lt - average latency in seconds
  • avg-ct - average connect time in seconds
  • stdev-rt - standard deviation for full response time
  • perc_90 - 90th percentile timing in seconds
  • min -minimum value of elapsed time in seconds
  • max -maximum value of elapsed time in seconds
  • bytes - response data size
  • throughput - total reqests made
  • succ -sucessful responses
  • fail - failed responses
  • rc.... - count of response codes like 200, 400, 502 etc..

individual report csv - 

  • timeStamp - in milliseconds since 1/1/1970 (epoch time)
  • elapsed - total time taken in milliseconds
  • label - testcase name / HTTP Sample Name
  • responseCode - e.g. 200, 404
  • responseMessage - e.g. OK
  • success - true or false
  • failureMessage - if any
  • bytes - number of bytes in the sample
  • sentBytes - number of bytes sent for the sample
  • grpThreads - number of active threads in this thread group
  • allThreads - total number of active threads in all groups
  • URL - actual url
  • latency - time to first response
  • connect - time to establish connection
  • IdleTime - number of milliseconds of 'Idle' time (normally 0)
